# 0x0C. C - More malloc, free
- 0-malloc_checked.c
	- contains a function, malloc_checked(), that allocates memory using malloc and returns a pointer to that memory
- 1-string_nconcat.c
	- contains a function, string_nconcat(), that concatenates the first n characters of s2 onto s1 and store in a new memory location
- 2-calloc.c
	- contains a functions that allocates memory for an array of nmemb elements of size bytes each and assigns all elements to 0
- 3-array_range.c
	- contains a function, array_range(), which creates an array of integers from min to max and returns pointer to array
