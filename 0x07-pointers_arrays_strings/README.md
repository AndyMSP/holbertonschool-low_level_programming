# 0x07. C - Even more pointers, arrays and strings
- 0-memset.c
	- Contains a function, \_memset(), that fills the first n bytes of the memory area pointed to by s with constant by b and returns a pointer to the memory area.
- 1-memcpy.c
	- Contains a function, \_memcpy(), that copies n bytes from memory area src to memory area dest and returns a pointer to dest.
